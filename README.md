# 🚀 Course Management System

A backend-focused **Course Management System** developed using **Java Spring Boot and PostgreSQL**. This project demonstrates backend development skills such as REST API design, database integration, and containerized deployment using Docker. The system allows management of courses and student enrollments while maintaining a scalable backend architecture. This project reflects my focus on building reliable backend systems and improving my full stack development skills with strong expertise in Java and Spring Boot.
